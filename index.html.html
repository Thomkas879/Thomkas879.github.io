CSS: Modelo de caja

Demostración del modelo de caja

El modelo de cuadro CSS es esencialmente un cuadro que envuelve cada elemento HTML. Consta de: bordes, relleno, márgenes y el contenido real.

Este texto es el contenido del cuadro. Hemos agregado un relleno de 50 px, un margen de 20 px y un borde verde de 15 px.
